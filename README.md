Required Concepts

Before starting this project, you should understand the following concepts:
Networking

    TCP sockets (bind, listen, accept)
    Non-blocking sockets
    Event-driven I/O (poll, epoll, kqueue)
    Connection handling with an event loop

HTTP

    HTTP/1.1 request and response structure
    Request line, headers, and body
    HTTP methods: GET, POST, DELETE
    Status codes and headers
    Chunked and unchunked transfers

Systems Programming

    File handling
    Process management (std::process::Command)
    Environment variables and pipes

CGI

    Basic understanding of the Common Gateway Interface
    Executing scripts and passing request data

Configuration

    Parsing configuration files
    Route configuration and server settings

Security

    Path traversal protection
    Request/body size limits
    CGI input sanitization

References

    RFC 9110 – HTTP Semantics
